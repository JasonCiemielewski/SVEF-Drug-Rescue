Role: You are an expert Bioinformatics Data Engineer and Cheminformatics specialist. Your goal is to build a high-fidelity dataset of "stranded" small-molecule drug assets for an Intellectual Property (IP) rescue pipeline.

Objective: Process the unzipped AACT (ClinicalTrials.gov) flat files to identify small-molecule drugs that reached Phase 2 or 3 but were terminated specifically for a lack of efficacy, ensuring they were not terminated for safety or toxicity reasons.

Context: The user is building the SVEF (Safety-Validated/Efficacy-Failed) dataset. These molecules are valuable because they have passed Phase 1 safety hurdles. The final output will be used for "Target Fishing" using Deep Learning models (like ICAN or MolTrans).

Data Source Specifications:

    AACT Files: Pipe-delimited (|) .txt files.

    Primary Tables: studies.txt and interventions.txt.

    Linking Key: nct_id.

Task Instructions:

Step 1: Data Ingestion & Joins

    Load studies.txt and interventions.txt using pandas. Use sep='|' and low_memory=False.

    Perform an inner join on nct_id.

Step 2: Initial Structural Filtering

    Filter for study_type == 'Interventional'.

    Filter for phase in ['Phase 2', 'Phase 3'].

    Filter for overall_status == 'Terminated'.

    Filter for intervention_type == 'Drug'.

Step 3: The "Safe but Futile" Logic (Regex Search)

    Clean the why_stopped column (handle NaNs and lowercase all text).

    Create a boolean mask for Efficacy Failures: Look for keywords: futility, efficacy, failed to meet primary endpoint, insufficient signal, lack of benefit.

    Create a boolean mask for Safety Failures: Look for keywords: toxicity, adverse events, safety concerns, side effects, maximum tolerated dose.

    Final SVEF Filter: Select rows where the Efficacy Mask is True AND the Safety Mask is False.

Step 4: Molecular Enrichment Placeholder

    Identify the unique intervention_name for all candidates.

    Prepare a function structure for a future API call to PubChem (PUG-REST) to retrieve SMILES, MolecularWeight, and LogP.

Step 5: Output & Quality Control

    Save the resulting dataframe to data/SVEF_candidates.csv.

    Print summary statistics: Total trials processed, number of efficacy failures identified, and the percentage of terminated trials that meet the SVEF criteria.

Constraints:

    Use Positron/Local file paths (assume data is in ./data_raw/).

    Ensure code is modular and commented for a graduate-level bioinformatics project.

    Exclude biologics; focus strictly on small molecules.
